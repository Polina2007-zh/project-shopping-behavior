# project-shopping-behavior
Этот проект представляет собой полный аналитический конвейер для исследования данных о покупательском поведении в интернет-магазине одежды и аксессуаров.Анализ данных в Python, Визуализацию и дашборд в Yandex DataLens, Публикацию проекта на GitHub .Датасет shopping_behavior.csv содержит информацию о 3900 покупках с характеристиками.
https://datalens.ru/g1ke2yrbh6180(ссылка на дашборд).
